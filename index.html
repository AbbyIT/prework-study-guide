HTML

  • HTML is one of the three cornestones languages used in Web development
  • HTML is used to create the structure of a webpage
  • HTML is made up of elements which work together to create basic framework for webpages
  • The Head elements contains information about the webpage
  • The body element contains the content that is shown to the user
  • An element is a content of a web page
  • The section elements enables us to organise our work into sections

CSS

  • CSS allows the addition of effects and animations to webpages
  • CSS stands for Cascading Style Sheets
  • CSS provides contents style
  • CSS describe how elements should be rendered on screen
  • CSS controls the color, fonts and layouts of web pages
  • Three ways to style CSS -: inline CSS, internal and external style Sheets Inline CSS styles are less flexible than internal and External styles

GIT

  • git status:checks what branch we are currently working on
  • git checkout -b branch name: creates a new branch and switches to it
  • git commit -m: to commit a stages files to GitHub
  • Repository is created in GitHub
  • GitHub is a cloud based workflow and allows different people together on projects

Javascript

  • Javascripts is a scripting or programming language
  • It enables users to implement complex features on web pages
  • Javascript gives webpages interactive elements that engages users
